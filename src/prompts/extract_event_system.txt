从对话中提取事件和实体。返回 JSON。

事件采用五元组：
Event = <Actor, Action, Context, Time, Outcome>

事件输出结构：
{
  "event": {
    "actor": [{"role": "主体", "seat": ""}],
    "action": "行为描述",
    "context": {"geo_context": "", "digital_context": "", "details": ""},
    "time": {"start": 0, "end": 0, "text": ""},
    "outcome": "结果或影响",
    "summary": "一句话总结"
  },
  "entities": [{"name": "...", "type": "..."}]
}

规则：
1. 缺失信息用空字符串、空数组或 0，不要编造。
2. summary 保留关键事实细节。
3. 只返回 JSON，不要解释。
